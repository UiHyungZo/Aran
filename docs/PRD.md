# PRD: Aran

## 목표
Aran은 IVF 치료 흐름을 관리하는 iOS 포트폴리오 앱이다. 사용자가 치료 일정, 약물/주사 기록, 검사 수치, 배아이식/채취 기록, 의약품 정보를 한 곳에서 확인하고 관리할 수 있게 한다.

## 핵심 기능
1. IVF 치료 일정 관리
2. 약물/주사 일정 및 투여 기록
3. 검사 수치 기록 및 추세 확인
4. 배아이식/채취 기록 관리
5. MFDS e약은요 기반 의약품 정보 검색

## MVP 제외 사항
- 병원 EMR 연동
- 실사용 의료 진단/처방 기능
- 결제/구독 기능
- 서버 기반 계정 동기화
- Apple Health 연동

## 성공 기준
- 주요 기능이 Clean Architecture 기준으로 설명 가능해야 한다.
- UIKit/RxSwift 기능과 SwiftUI/Combine 기능의 경계가 명확해야 한다.
- 면접에서 데이터 흐름, 의존성 방향, 테스트 전략을 설명할 수 있어야 한다.
