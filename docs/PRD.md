# PRD: Aran v13.0

## 목표

Aran은 IVF 치료 흐름을 관리하는 iOS 포트폴리오 앱이다.
사용자가 치료 일정, 약물/주사 기록, 검사 수치, 배아이식/채취 기록, 의약품 정보를
한 곳에서 확인하고 관리할 수 있게 한다.

---

## 앱 구조 — 5탭

| 탭 | 기술 스택 | 역할 |
|----|-----------|------|
| 📅 캘린더 | SwiftUI + Combine | 날짜 기반 모든 정보의 허브. 2단계 바텀시트 구조 |
| 🗂 시술 기록 | SwiftUI + Combine + Swift Charts | 차수별 채취/이식/PGT 이력 관리 및 차트 |
| 💊 약/주사 | UIKit + RxSwift | 복용 약 관리, 알림 설정, 약 검색 register 모드 |
| 🧪 검사 | UIKit + RxSwift + Swift Charts | 검사 수치 입력, 항목별 목록, 트렌드 차트 |
| 🔍 약 정보 | SwiftUI + Combine | 식약처 API 약 검색 browse 모드, 상세 정보 |

---

## 캘린더 탭 — 2단계 바텀시트 구조

| 단계 | 화면 | 설명 |
|------|------|------|
| 0단계 | 캘린더 메인 | 월간 달력. 날짜별 도트 표시. 날짜 탭 시 1단계 시트 |
| 1단계 | 날짜 요약 시트 | 해당 주 7일 + 선택 날짜 모든 기록 요약. 항목 탭 시 2단계 시트 |
| 2단계 | 항목별 입력 시트 | 실제 데이터 입력. 1단계 시트는 뒤에 흐리게 유지 |

### 1단계 시트 섹션

| 섹션 | 기록 있을 때 | 기록 없을 때 | 탭 시 동작 |
|------|-------------|-------------|-----------|
| 병원 일정 | 일정 종류 + 메모 | "일정 없음" | 2단계: 병원 일정 입력 |
| 복용 약 | 약 이름 pill 목록 | "복용 약 없음" | 약/주사 탭으로 이동 |
| 감정 일기 | 이모지 + 텍스트 미리보기 | "기록 없음" | 2단계: 감정 일기 입력 |
| 검사 수치 | 항목·수치 요약 | "기록 없음" | 2단계: 검사 수치 입력 |
| 생리 시작일 | 생리 시작일로 표시 | "오늘로 기록하기" 버튼 | 2단계: 생리 주기 입력 |

### 2단계 시트 입력 내용

| 시트 | 입력 내용 |
|------|-----------|
| 병원 일정 | 날짜(자동) · 일정 종류(내원/채혈/초음파) · 메모 |
| 감정 일기 | 이모지 선택(😊😢😰😤🥰) · 텍스트(최대 500자) |
| 검사 수치 | 항목 선택 · 수치 · 단위 · 측정일 · 메모 |
| 생리 주기 | 생리 시작일(자동) · 주기 길이(기본 28일) → 배란 예정일 자동 계산 |

## 핵심 기능 (탭별)

### TAB 1 · 📅 캘린더

- 월간 달력 뷰 — 날짜별 색상 도트, 월 이동
- 날짜 탭 → 1단계 요약 시트
- 병원 일정 / 감정 일기 / 검사 수치 / 생리 주기 2단계 입력 시트
- SwiftData 저장

### TAB 2 · 🗂 시술 기록

- 차수 목록 — 차수별 카드(채취/수정/동결 개수, 이식 결과 요약)
- 채취/이식 입력 — 차수·개수·등급·동결/신선
- 이식 결과 기록 — 이식일·등급·개수·결과(성공/실패/진행중)
- PGT / 반착검사 기록
- Swift Charts — 차수별 채취→수정→동결→이식 흐름 Bar/Line Chart

### TAB 3 · 💊 약/주사

- 약 목록 — UITableView, 복용 시간, 체크 상태
- 스와이프 액션 — 중단/삭제
- 약 등록 폼 — DrugSearch register 모드로 자동 입력
- UserNotifications 알림 — 등록/수정/삭제/ON/OFF
- 알림 미리보기

### TAB 4 · 🧪 검사

- 수치 입력 폼 — 항목 선택 + 수치 + 단위 + 날짜 + 메모
- 수치 목록 — 항목별 최신 수치 + 증감 TrendBadge
- 수치 히스토리 — 항목별 시간순 목록
- Swift Charts Line Chart — 수치 변화, 정상 범위 레퍼런스 라인

### TAB 5 · 🔍 약 정보

- 약 이름 검색 — Combine .debounce(0.3s) → e약은요 API
- 검색 결과 목록
- 약 상세 정보 — 효능/용법/주의사항/경고 배너
- 약 추가 연동 — "이 약 추가하기" → MedicationFormVC
- 최근 검색어 — UserDefaults 저장

## 성공 기준

- 주요 기능이 Clean Architecture 기준으로 설명 가능해야 한다.
- UIKit/RxSwift 기능과 SwiftUI/Combine 기능의 경계가 명확해야 한다.
- 면접에서 데이터 흐름, 의존성 방향, 테스트 전략을 설명할 수 있어야 한다.
- UseCase / Repository / ViewModel / UI 전 레이어 테스트가 존재해야 한다.
- 다크모드가 완전 대응되어야 한다.
