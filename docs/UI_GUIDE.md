# UI Guide

## 목적

Claude Code가 임의 스타일로 구현하지 않도록 Aran 앱의 화면 톤, 컬러 시스템,
컴포넌트 패턴, 금지 패턴을 정의한다.

---

## 기본 원칙

- 의료/치료 기록 앱이므로 차분하고 명확한 UI를 우선한다.
- 정보 계층을 분명히 하고, 장식보다 가독성을 우선한다.
- Empty, Loading, Error 상태를 화면별로 반드시 구현한다.
- 다크모드 완전 대응 — Light/Dark 두 벌 색상 정의 필수.

---

### 상태 배지 색상

| 상태 | 배경 | 텍스트 |
|------|------|--------|
| 진행중 | `#FAEEDA` | `#633806` |
| 성공 | `#E1F5EE` | `#085041` |

### 트렌드 배지 (검사 수치)

| 방향 | 배경 | 텍스트 | 의미 |
|------|------|--------|------|
| ↑ 상승 | `#FCEBEB` | `#A32D2D` | 증가 |
| ↓ 하락 | `#E1F5EE` | `#085041` | 감소 |

> 트렌드 방향의 좋고 나쁨은 항목마다 다르므로 색상은 방향만 표시한다.

### 다크모드 대응

- `Color(UIColor.systemBackground)` 등 Adaptive Color 전면 적용
- Assets.xcassets에 Light/Dark 두 벌 정의
- SwiftUI: `colorScheme` 환경 변수 활용
- UIKit: `traitCollectionDidChange` 대응
- Swift Charts 다크모드 색상 별도 정의 필수

## 컴포넌트 규칙

### 공통
- 치료 일정, 약물, 검사 기록은 카드 기반 리스트를 우선한다.
- 주요 CTA는 화면당 1개를 원칙으로 한다 (탭별 대표 색상 적용).
- 위험/삭제 액션은 확인 단계(ActionSheet 또는 Alert)를 둔다.
- 드래그 핸들(20×2pt, 회색)을 바텀시트 상단에 배치한다.

### 캘린더 탭 (SwiftUI)
- 2단계 바텀시트 구조 유지 — 1단계 시트는 2단계 진입 시 25% opacity로 뒤에 유지
- 날짜 선택 시 Pink 원형 배경 (today: Pink fill, selected: Pink border)
- 각 섹션 탭 → 해당 2단계 시트 진입

#### 복용 약 체크박스 (1단계 시트)
- 약 이름 pill 옆에 원형 체크박스 배치
- 체크 시: Purple(`#534AB7`) fill + 흰색 내부 원
- 미체크 시: 회색 테두리 원
- 탭 = MedicationLog isTaken 토글 (2단계 진입 없음)

#### 병원 일정 종류 chip (2단계 시트)
- 내원 / 채혈 / 초음파 칩 — 복수 선택 가능
- 선택 시: `#1a1a1a` 배경 + 흰색 텍스트
- 미선택 시: `#f5f5f5` 배경 + `#888` 텍스트 + `#ddd` 테두리

### 시술 기록 탭 (SwiftUI + Combine)
- 차수 카드 — 상단 차수명 + 상태 배지, 하단 수치 요약
- Swift Charts Bar Chart — Teal 계열 그라데이션 (`#9FE1CB` → `#0F6E56`)
- 배아 등급 chip: 선택 시 `#E1F5EE` 배경 + `#085041` 텍스트 + `#1D9E75` 테두리

### 약/주사 탭 (UIKit + RxSwift)
- 체크박스: 완료 시 Purple(`#534AB7`) fill + 흰색 내부 원
- 스와이프 액션: 중단 `#888888`, 삭제 `#A32D2D`
- 자동 입력 필드 배경: `#EEEDFE` (Purple 계열 연한 배경)

### 검사 탭 (UIKit + RxSwift)
- 수치 텍스트: Blue(`#185FA5`) bold
- 입력 필드 활성 배경: `#E6F1FB`
- Swift Charts Line Chart: Blue 계열 (`#B5D4F4` → `#185FA5`)
- 커스텀 항목 추가 셀: "+" 아이콘 + "직접 추가" 텍스트

### 약 정보 탭 (SwiftUI + Combine)
- 검색 결과 배지: `#EAF3DE` 배경 + `#27500A` 텍스트
- 성분 배지: `#EAF3DE` 배경 + `#27500A` 텍스트
- 전문의약품 경고 배너: `#FAEEDA` 배경 + `#633806` 텍스트

## 타이포그래피

- 시스템 폰트 사용 (`-apple-system`, SF Pro)
- AranFont 커스텀 폰트 정의는 `DesignSystem/Fonts.swift` 참고
- 수치 표시 (검사, 차수): `font-weight: 500`, 크기 14~16pt

---

## 금지 패턴

- 과도한 glass morphism
- 의미 없는 그라데이션 텍스트
- 의료 정보 화면에서 과도한 애니메이션
- ViewController/View 내부에서 임의 색상 hex 하드코딩 (반드시 Assets 또는 DesignSystem/Colors.swift 사용)
- 탭 대표 색상을 다른 탭에서 사용 (예: 검사 탭에서 Pink CTA 버튼)
